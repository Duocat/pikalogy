pikalogy

Pikachu theme for Terminology

Terminology is the genius terminal emulator developed by the Enlightenment Foundation (http://web.enlightenment.org/p.php?p=index&l=en).

It is available for all GNU/Linux distributions (I guess) and for Mac OS X (currently only with the X window manager).

Terminology can be downloaded from here: http://web.enlightenment.org/p.php?p=about/terminology&l=en but the development version is stable enough to be used. To get the development version, just execute:
git clone http://git.enlightenment.org/apps/terminology.git

Usage

  • Make sure terminology is installed (see download link above)
  • In the directory of the pikalogy theme
    • make to compile the Edje file
    • make play to launch terminology with the pikalogy theme
    • sudo make install to install the files
    • sudo make uninstall to remove installed files
